HTML table code reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all table elements
<thead> Table head The set of rows defining the column headers in the table
<tbody> Table body The set of rows containing the actual table data
<tr> Table row The table row container
<td> Table data The data within a table row
<tfoot> Table foot The row containing the data at the foot of a table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row span Defines how many rows a td element should span
